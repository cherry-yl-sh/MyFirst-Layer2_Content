# 3.2 进一步压缩

数据压缩是个远早于区块链就存在的技术，除了前面对交易的定制化压缩之外，还有许多压缩算法能帮助进一步压缩数据空间。

比如我们生活中往往都接触过zip、rar、7z压缩包，它们就是使用了压缩算法将各种文件的体积减小。Optimism的Zlib压缩算法、Arbitrum的Brotli压缩算法都能起到类似的作用。

【类似于ZIP、RAR、7z的图】

我们可以看到各种Layer2 Rollup方案相较于以太坊主网，都取得了不错的压缩效果。
